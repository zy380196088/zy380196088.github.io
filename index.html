React简介

对 React.JS 的认识 及其优点

React 并不是一个完整的 MVC 框架,最多可以认为是 MVC 中的 V(View);
React 不是一个新的模板语言, JSX 只是一个表现,没有 JSX 的 React也能工作.

React本质上是一个”状态机”,可以帮助开发者管理复杂的随着时间变化的状态.
React只关心两件事情:

  1. 更新 DOM
  2. 响应时事件
